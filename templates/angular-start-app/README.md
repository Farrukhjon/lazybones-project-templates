Overview
----------
AngularJS based, front-end application template.

## Directory structure template is modular based, for enterprise app.
```
├── app
│   ├── about
│   │   ├── aboutView.html
│   │   ├── controllers
│   │   ├── directives
│   │   ├── filters
│   │   └── services
│   ├── app.js
│   ├── common
│   │   ├── commonView.html
│   │   ├── controllers
│   │   ├── directives
│   │   ├── filters
│   │   └── services
│   └── home
│       ├── controllers
│       ├── directives
│       ├── filters
│       ├── homeView.html
│       └── services
index.html

```
